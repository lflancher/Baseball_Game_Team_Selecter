# Baseball_Game_Team_Selecter
This is a not for profit program to select teams for a popular baseball video game and will allow users to choose between true random and weighted teams.
Designed primarily as a way to quickly select teams for a certain popular baseball game that will not create bias for selecting one captain. Teammates can be either completely random or they can be "weighted". Weighted essentially means that both teams will get a tri-chemistry at least once so that mechanics that allow one to catch homeruns will always be available. The algorithm essentially looks through the first four characters in a team (this includes the captain) and then adds an appropriate amount of S-ranked captains to the team that has fewer, before selecting the rest of the teammates.
